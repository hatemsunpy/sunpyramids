# Contracts

**Feature**: Image Optimization (007-image-optimization)

No external interface contracts are required for this feature.

Image Optimization is a purely internal frontend enhancement that:
- Modifies Vue component image attributes (`decoding`, `loading`, `fetchpriority`, `width`, `height`)
- Converts static PNG assets to WebP format
- Updates internal file references

There are no new public APIs, command schemas, endpoints, or external system integrations introduced by this feature.
